# Test Site Blog

Experimental TypeScript blog platform based on Next.js with content and deployment tooling.

## Purpose

This repository is used to prototype and validate blog workflows, including content authoring, site configuration, and deployment automation.

## Expected Experience

- Run locally and iterate on blog features quickly
- Author posts in MDX
- Configure metadata, layouts, and publishing behavior
- Deploy with a repeatable pipeline

## Getting Started

```bash
yarn install
yarn dev
```

## Key Areas

- `app/` routes and application pages
- `data/` blog content and metadata
- `components/` reusable UI pieces
- `scripts/` build and post-build tooling
